# Wien-Bridge-Oscillator-with-Automatic-Gain-Control-AGC-
This repository contains the design, simulation, and PCB layout files for an analog sinusoidal oscillator featuring JFET-based amplitude stabilization. The project was developed with a strong focus on signal purity, thermal reliability, and industry-standard manufacturability.
 Circuit Design & Analysis
Core Architecture: Analog Wien Bridge sinusoidal oscillator.

Frequency Range: Adjustable between 0.5 KHz and 3 KHz.

Amplitude Stabilization: Implemented an Automatic Gain Control (AGC) loop using a JFET.

Reliability: Conducted Worst-Case Analysis to ensure stable operation across a 0°C to 70°C temperature range.

 Simulation & Validation
Performed comprehensive transient and AC sweep simulations using PSpice.

Validated oscillation start-up conditions and verified overall signal purity before moving to the layout phase.

 PCB Layout & Manufacturing Standards
Designed a compact, double-layer printed circuit board (PCB) using OrCAD PCB Editor.

Compliance: Strictly adhered to IPC-2221A standards for track routing, clearance spacing, and Surface Mount Technology (SMT) footprint design.

Production Ready: Generated standard manufacturing files, including Gerber RS-274X and Excellon drill files, ensuring direct compatibility with PCB fabrication houses.
